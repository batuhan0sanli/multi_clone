Installation
============

Requirements
------------

* Python 3.8 or later (https://www.python.org/downloads/)
* RClone (https://rclone.org/downloads/)

Installation
------------

Install from PyPI:

.. code-block:: bash

      python3 -m pip install rclone-manager

Install in Terminal:

.. code-block:: bash

      git clone
      cd rclone-manager
      python3 -m pip install .
